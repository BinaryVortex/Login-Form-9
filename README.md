# Login Form

A clean, responsive login form built using HTML, CSS and JavaScript.

![Login form screenshot](./Screenshot%202024-05-08%20195113.png)

## Description

This repository contains a simple, modern login form UI that demonstrates layout, styling, and basic client-side validation using vanilla HTML, CSS, and JavaScript. It's ideal as a small component for practice, portfolio projects, or as a starting point for integrating real authentication.

## Features

- Responsive layout that adapts to mobile and desktop screens
- Clean, accessible form fields for email/username and password
- Lightweight client-side validation (required fields)
- Smooth visual styling with CSS

## Demo / How to use

1. Clone or download this repository.
2. Open `index.html` in your web browser (double-click the file or serve it with a local web server).

Alternatively, run a simple local server from the project directory (Python):

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

## Files

- `index.html` — markup for the login form
- `style.css` — styling for layout, colors, and responsiveness
